This is the repository for the PSU Chem-E Car
The intended purpose is to document Arduino code from past years

All files can easily be copied and uploaded to Arduino
Alternatively you can clone the entire repository

Once year is complete please upload your files as explained below!

Upload your competition-ready code as well as any testing file you deem useful
EDP Circuit Diagram should be uploaded as its own file
Make sure to include the year in the filename

Descriptions for the use of each file should be placed at the top of the code
Descriptions should include...
- Stopping Mechanism's design so future years know the use cases
- Explanation of code 
- Any other useful information to know

Thank you!
