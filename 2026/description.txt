Objective:
	This year used an "elephant toothpaste" stopping mechanism, and this code controls 
	the arduino according to a photoresistor reading values directly from an attached laser
	in an eclosed box. The reaction fills a chamber with a mostly opaque fluid, blocking the laser, signaling the
	arduino to shut off the motor.

Key Details:
	The light_level shutoff value may vary from what is currently present. Run wet tests to measure
	a consistent level with no obstruction and with obstructions to find a reasonable shutoff level.
